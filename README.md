## `> ./install.sh`

### Option 1 — Docker (Recommandé)

```bash
# Clone
git clone https://github.com/Durrell-229/NEXUS-HACKER-PLATEFORM.git
cd NEXUS-HACKER-PLATEFORM

# Configure
cp .env.example .env
# → Edit .env with your SECRET_KEY, DATABASE_URL etc.

# Launch
docker compose up --build

# Access
# Frontend  →  http://localhost:5173
# API       →  http://localhost:8000/api/
# Admin     →  http://localhost:8000/admin/
```

### Option 2 — Manuel

```bash
# Frontend
cd frontend && npm install && npm run dev

# Backend (new terminal)
cd backend
python -m venv .venv && source .venv/bin/activate  # or .venv\Scripts\activate on Windows
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

## `> ./deploy.sh --target=cloud`

### 🌐 Frontend → Vercel

```bash
cd frontend
npx vercel --prod
# Set env vars: VITE_API_URL → your Render backend URL
```

### ☁️ Backend → Render (Blueprint)

```
1. Push to GitHub
2. render.com → New → Blueprint
3. Connect: github.com/Durrell-229/NEXUS-HACKER-PLATEFORM
4. Render reads render.yaml → provisions:
   ✓ Django web service (Docker)
   ✓ PostgreSQL database (free)
   ✓ Redis instance (free)
5. Shell → python manage.py createsuperuser
```
